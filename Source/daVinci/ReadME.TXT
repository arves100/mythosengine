 DESCRIPTION

 This library builds a 3D scene manager on top of the basic 3D primitive
 support provided by DEGAS.  This includes basic and advanced drawables
 such as sprites, meshes, terrain, particle systems, and portals;
 partitioning and retained-mode operation; and animation support.
